# 文件格式与 I/O

<div class="deck-rule"></div>

<div class="deck-lead">
  文件格式用自定义 <code>.vgjson</code>。JSON 字段语义明确：<code>id / type / geometry / style / transform / zValue</code>，顶层结构为 <code>version + canvas + shapes</code>，定义稳定的反序列化契约。
</div>

```json
{ "version": 2, "canvas": { "width": 1200, "height": 800 }, "shapes": [ ... ] }
```

<div class="diagram-caption">由 <code>shapeDataToJson</code> 与 <code>shapeDataFromJson</code> 统一编解码。</div>
