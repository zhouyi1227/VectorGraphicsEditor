<div class="deck-lead">
  文件格式用自定义的 <code>.vgjson</code>。JSON 字段有明确的语义：<code>id / type / geometry / style / transform / zValue</code>，单 shape 结构和顶层 <code>version + canvas + shapes</code> 一起定义出稳定的反序列化契约。
</div>

```json
{
  "version": 2,
  "canvas": { "width": 1200, "height": 800 },
  "shapes": [ ... ]
}
```

<div class="diagram-caption">单个 shape 细分为 <code>id / type / geometry / style / transform / zValue</code>，由 <code>shapeDataToJson</code> 与 <code>shapeDataFromJson</code> 统一编解码。</div>
